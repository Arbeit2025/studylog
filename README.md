# studylog

studylog は、Flask と SQLite で作られたシンプルな学習ログアプリです。
  
ブラウザから学習内容を記録し、一覧表示し、不要になったら削除できます。  
あわせて、今月の合計学習時間や、平均集中度・平均理解度も確認できます。

小さくてシンプルな学習ログを目的としています。


Single-file Flask + SQLite study log app.

## Features
- 学習ログの保存
- 学習ログ一覧表示
- 学習ログ削除
- 今月の合計学習時間表示
- 平均集中度の表示
- 平均理解度の表示

## Run (local)
pip install -r requirements.txt
python app.py
